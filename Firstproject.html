HTML Semantic Tags Cheat Sheet

Semantic Non-Media Tags

Tag Name Descritpion
<header> Header Container for header content.
<nav> Nav Container for navigation content.
<main> Main Container for dominant content of page.
<section> Section Container for content with the same theme as main.
<article> Article Container for articles, blogs, comments, etc..
<aside> Aside Container for additional content, but it is not connected to main.
<footer> Footer Container for footer content.

Semantic Media Tags

Tag Name Descritpion
<figure> Figure Container for images.
<figcaption> Fig Caption Caption of an image.
<audio> Audio Container for audio content.
<video> Video Container for video content.
<embed> Embed Container for any other type of media like gifs.
